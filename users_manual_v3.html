Game Cards

Each card on the grid shows:

  • Game title and court location
  • Format badge (Full Court / Half Court / 3-on-3) and skill level
  • Schedule — day, date, and time in 12-hr format (e.g., Thu, Mar 19 @ 7:00 PM)
  • Roster bar — shows current players / max slots with a fill progress bar
  • Entry fee — Free or a peso amount
  • JOIN RUN button — changes to CONFIRMED if you've already joined
🛜
Live Updates! New games posted by other users appear automatically. A 🏀 New game just posted! banner slides in and fades out after 5 seconds — no page refresh needed.

Joining a Run

From the Dashboard

  1. Find a game card on the dashboard.

  2. Click JOIN RUN on the card.

  3. If the game has House Rules, a modal appears — read it and click I AGREE & JOIN THE RUN.

  4. If the game is full, the button changes to JOIN THE WAITLIST — you'll move up automatically if a spot opens.

  5. Once joined, the button changes to ✓ CONFIRMED.

ℹ️
Already joined? The dashboard detects your join status on page load — you'll see CONFIRMED immediately without needing to click anything.

From the Game Detail Page

  1. Click anywhere on a game card to open the full detail page.

  2. Review the full game info — format, rules, roster, location.

  3. Click the large + JOIN THE RUN button in the hero section.

Host Controls — Roster Management
/game/[id] (Host)

Per-Player Roster Actions

ButtonAction
✅ Green checkmarkMark player as Checked In (present on court)
❌ Red XMark player as Absent (lowers their reliability score)
🗑️ Trash iconKick player from the squad entirely
RESETRevert a marked player back to Joined

Waitlist Promotion

If the main roster has space and players are waiting: click SQUAD UP next to a waitlisted player to move them into the main roster.

Team Generator

  1. Once your squad is confirmed, scroll to the Team Generator section.

  2. Click Generate Teams.

  3. The system splits players into 2 balanced teams by skill level and position.

  4. Team assignments are saved and visible to all players on the roster.

Game Lifecycle

ActionWhen to useEffect
✅ COMPLETE RUN After the game ends Unlocks stat logging for all checked-in players. Game marked as completed.
❌ CANCEL RUN If the game won't happen Requires a reason. All joined players will see the cancellation reason on the game card.
⚠️
Cancellation — once a game is cancelled, it cannot be uncancelled. Always use this option only when necessary.

QR Check-In System

OpenCourt uses QR codes to confirm player attendance on game day.

📱

As a Player — Show Your Pass

Open the game detail page → click 📱 SHOW CHECK-IN QR → a "Player Pass" QR code appears. Show this to the host when you arrive.

📷

As a Host — Scan Mode

Open the game detail page → click 📷 SCAN MODE in the roster section → point camera at each player's QR code. They're marked Checked In ✅ instantly.

opencourtph.vercel.app/game/[id] — QR Code
📱
Player Pass QR Code
/game/[id] — Check-In QR
⚠️
Attendance affects your reliability score. Being marked Absent without notice will lower your score. Try to inform the host if you can't make it.
11

Reliability Score

Your Reliability Score starts at 100% and reflects how dependable you are as a player. It's visible to hosts when they view your profile.

EventImpact
✅ Checked In by hostScore maintained / improves over time
❌ Marked Absent by hostScore decreases
Consistent attendance streakScore recovers and grows
🎯
Aim for above 80%. Hosts can see your reliability score and may prefer to invite players with high scores when games fill up.
⭐
100% Reliable badge appears on your hosted games' "Hosted By" card when your score is perfect — a great signal of trustworthiness to potential players.
